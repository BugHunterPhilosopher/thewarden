Download Tor  [Optional]  [Required with the Dojo]

When using Tor, traffic is routed through multiple servers and encypted in each step.

If enabled, the application will use Tor to send price requests using onion routing.

Please note that using Tor will slow down the application.

Download Tor



Get an Alphavantage API Key  

An Alphavantage API key is suggested to download historical prices. We use a couple of other services to get prices (that do not require an API key) but Alphavantage is another reliable source of prices for stocks, currencies and crypto.
Click here to get a free API key. Then input it below to test.
