Getting Started


1. Get an Alphavantage API Key  [Required]

An Alphavantage API key is required to download historical prices.
Click here to get a free API key. Then input it below to test.

Tor can be setup later for more private price requests



2. Download Tor  [Optional]  [Required with the Dojo]

When using Tor, traffic is routed through multiple servers and encypted in each step.

If enabled, the application will use Tor to send price requests using onion routing.

Please note that using Tor will slow down the application.

Download Tor



3. Step into the Dojo   [Optional]  

Use your own Bitcoin Node to track wallet and address balances.

Import transactions straight from the blockchain into your portfolio.

Install the Dojo

Already running a Dojo?

Awesome! Just enter your Dojo configuration below.

Your Dojo onion address can be found by running the command ./dojo.sh onion
The API Key was created at the Dojo initial setup (at the docker-node.conf file)


4. Import Transactions   [Optional]  

You may enter your transactions:

1. Manually: Click at the sidebar.

2. Import a CSV file: Click at the sidebar.

3. Import from your Node with the Dojo: If you have a Dojo setup, click here.
